Находится в del6 virtualenv

Эта версия запускается так:

. /home/ai/del6/bin/activate
cd ~/del6/ray/rllib/examples
python3.5 custom_env_3.py

В этой версии оператор успешно обучает DQN подавать правильную команду "right"  (а не "left") (или наоборот).
В пределах 5 минут (на каждый опыт) четыре раза подряд справилась с этой задачей.

На вход подаётся изображение с веб-камеры.
Реворд DQN = средняя освещенность камеры.
На команду "right" оператор открывает камеру, на команду "left" закрывает.
